## 💁‍♀️ 프로젝트 개요
- 본 프로젝트는 멋쟁이 사자처럼 대학 10기 해커톤을 통해 제작되었습니다.
- 아!끼자는 소비자를 위한 공동 구매 플랫폼입니다.
- 공동 구매를 원하는 소비자는 공동 구매 할 물품을 등록하여 다른 소비자와 함께 공동 구매를 하거나 다른 소비자가 등록한 공동 구매 물품을 선택하여 공동 구매에 참여할 수 있습니다.
